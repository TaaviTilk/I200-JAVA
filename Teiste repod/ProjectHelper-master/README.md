ProjecHelper

Tegemist on aine "Programmeerimise algkursus Java baasil" projektiga, mille raames kirjutan erinevate projektidega tegelemisel kasutatavat abimeest.
Programmi täpne olemus selgub selle tegemise käigus, kuid üldjoontes peaks see võimaldama sammude ülesmärkimist, märkmete tegemise võimalust ja meeldetuletusi.
Lisaks on plaanis luua ka visuaalselt korralik kasutajaliides.
